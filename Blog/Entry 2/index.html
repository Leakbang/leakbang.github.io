    Why Github

    An Experiment

    So Github is a very good platform for developers in general. You can share your creation with others in an open-source manner and other people can also contribute to your creation, expanding it. Github also offers to host static websites for free and the website is handled like other open source projects on Github. It means that people can contribute to it, by adding new entries, translations, fixing typos/error and etc.

    This is an extremely simplistic website with bare minimum html and css. I've never created a website in such way as my other projects were much more demanding but there is a certain charm to how this works and feels like. So I've created this website as a sort of personal blog to openly share my thoughts and recommendations on various different subjects.

    Feel free to contribute to this blog in any way you'd like.
