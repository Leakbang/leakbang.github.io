    Temple OS

    Link: Click Here
    Comments: Perhaps the most interesting operating system. Great place for coders to experiment. A ton of exclusive applications and games to try out. It's recommended to run it in a VM.
    License:Free, Open Source

    Goat Counter

    Info: Easy web analytics. No tracking of personal data. GoatCounter is an open source web analytics platform available as a hosted service or self-hosted app. It aims to offer easy to use and meaningful privacy-friendly web analytics as an alternative to Google Analytics or Matomo.
    Link: Click here
    Comments: It's an incredibly good utility and it's a very good alternative to Google Analytics. It is free for non-commercial use and it can be self hosted as well. Another good thing about it is that it provides pixel tracking analytics for places where JavaScript is disabled.
    License:Free (For Non-Commercial use), Open Source

    Topaz Software Suite

    Info: A suite of AI powered softwares useful to 2D and 3D artists.
    Link: Click here
    Comments: Really amazing pieces of software. They use machine learning AI to do magic. They are expensive, but based on my experience, they do the job far better than other softwares in the competition.
    License:Paid

    WMware Workstation

    Info: Extremely agile software for running virtual machines on Windows, Linux, and mac.
    Link: Click here
    Comments: From my experience, it's the best software in the competition. It is packed with features and runs much faster than other VM software like virutalbox. It costs a bunch, but if you use VMs professionally it's definitely the way to go. For personal use, there is VMware workstation player which is the free, lite version. Still better than VitualBox.
    License:Free (For Non-Commercial use)

    ZeroNet

    Info: Open, free and uncensorable websites, using Bitcoin cryptography and BitTorrent network
    Link: Click here
    Comments: A decentralized network of websites and webservices, uncensored. It being uncensored means that a lot of questionable content can be found on it but overall, it's a very good piece of software. Everything is routed through TOR.
    License:Free, Open Source

    Tails OS

    Info: Tails is a portable operating system that protects against surveillance and censorship.
    Link: Click here
    Comments: Highly recommended. Probably the most secure and anonymous way to use a computer. I heavily recommend to install it to a bootable USB rather than running it in a Virtual Machine. While it can be used as a daily driver, it is not it's main purpose. I cannot recommend this enough. You can access the internet and do your tasks anonymously on any machine. Everything is routed through TOR.
    License:Free, Open Source
